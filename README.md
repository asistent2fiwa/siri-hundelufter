# 🐕 Siri - Hundelufter Website

## Filer

- `index.html` - Hjemmesiden

## Sådan uploades til GitHub Pages

1. Opret et nyt GitHub repository (f.eks. "siri-hundelufter")
2. Upload `index.html` til repoet
3. Gå til Settings → Pages
4. Vælg "main" branch og klik Save
5. Din side vil være tilgængelig på: `dit-brugernavn.github.io/siri-hundelufter`

## Rediger telefonnummer

Åbn `index.html` og find:
```html
<a href="tel:12345678" class="contact-btn">📞 Kontakt Siri</a>
```

Ændr `12345678` til det rigtige nummer.

## Tilføj billeder

Find "placeholder" sektionerne og erstat med rigtige billeder:
```html
<img src="dit-billede.jpg" alt="Siri og hund">
```
